D. 旅行（travel）

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
转到题库 提交记录 题解 返回比赛

题目描述

志坚者，不以山海为远.
户外攀登的有形之山，尚且存在有形之路，而学术科研的无形之山，则需要在无人区独自披荆斩棘。高中三年风雨兼程， 早已循滔乎此，望本期理性，加以努力。前进而勿顾后，背黑暗而向光明，为世界进文明为人类造幸福.
学术之巅不止一座，通往成功之路不止一条。如今意志坚定的 望着漫漫长路，打算竹杖芒鞋越重山. 已知有 个学术发展节点，不同发展节点中有 单向条发展道路，每到达第 个发展节点， 就要花费 额外时间去掌握该技能，他终将到达学术之巅 (一定可以到达并且学习完 N 号节点的知识)。

输入格式

第一行两个整数 分别表示发展节点的个数和发展道路的数量。
第二行共 个数，表示在第 个发展节点掌握技能所花费的时间。
从第三行开始，到第 行，每行 个数 表示单向发展道路的起点，终点和 到 所需时间。

输出格式

一行一个整数，表示你从节点 出发，学习完第 N 给知识所花费的最小时间。

样例

样例输入复制

3 2
0 0 0
1 2 1
2 3 1

样例输出复制

2
