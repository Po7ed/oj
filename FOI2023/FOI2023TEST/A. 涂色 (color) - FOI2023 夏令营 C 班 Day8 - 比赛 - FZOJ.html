A. 涂色 (color)

内存限制：32 MiB 时间限制：4000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
转到题库 提交记录 题解 返回比赛

题目描述

个葫芦排成一排，从左到右依次编号为 每次给定 个整数 乜乜就会从葫芦 开始到葫芦 依次给每个葫芦涂一次颜色。但是 次以后，乜乜已经忘记了第 个葫芦已经涂过几次颜色了，你能帮他算出每个葫芦被涂过几次颜色吗？

输入格式

共若干测试实例，每个测试实例第一行为一个整数 . 接下来的 行，每行包括 个整数 当输入结束。

输出格式

每个测试实例输出一行，包括 个整数，第 个数代表第 个葫芦总共被涂色的次数。

样例

样例输入复制

3
1 1
2 2
3 3
3
1 1
1 2
1 3
0

样例输出复制

1 1 1
3 2 1
